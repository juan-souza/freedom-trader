# BotCrypto

###

### Dependencias FrontEnd(Angular9)

- npm install -g @angular/cli

  - Instalação do angular global

- Toaster (ngx-toastr)

  - npm install ngx-toastr --save

- Decore JWT
  - npm install @auth0/angular-jwt

**Comandos**

- ng new (Nome do projeto)
  - Cria o projeto
- ng serve --open
  - Start a aplicacao
- ng generate component (ng g c (nome))
  - Cria os componentes
- ng generate model (
